## 🚀 Featured Projects

### 🧠 [AnalytIQ — Natural Language BI Assistant](https://github.com/Sahaj-Chakka/AnalytIQ)
> RAG-powered assistant that lets business users query operational & sales data in plain English — no SQL required

- Built with **LangChain + OpenAI API**, embedded docs into a **FAISS vector store**, served via **Streamlit**
- Validated against 50 real business questions → **91% response accuracy**
- `Python` `LangChain` `OpenAI API` `FAISS` `Streamlit` `SQL`

---

### 📉 [RetainIQ — Churn Prediction & Revenue Forecasting](https://github.com/Sahaj-Chakka/RetainIQ)
> ML pipeline across 500K+ transactions to predict churn and forecast revenue

- Benchmarked 7 models — landed on **XGBoost (AUC: 0.87)** for churn, **Prophet (RMSE 12% below baseline)** for forecasting
- Identified top 18% highest-risk customers driving majority of projected churn
- `Python` `XGBoost` `LightGBM` `Prophet` `LSTM` `SQL` `Tableau`

---

### 📊 [PulseBoard — Real-Time Operational Analytics Dashboard](https://github.com/Sahaj-Chakka/PulseBoard)
> Automated ETL + anomaly detection system monitoring 15+ KPIs in near real-time

- Cut reporting latency by **60%**, saved **8–10 hrs/week** of manual review via statistical process control scripts
- `AWS S3` `Redshift` `Python` `SQL` `Power BI`

### 🏗️ [IntegriSphere — Cloud Data Lakehouse & Self-Serve Analytics](https://github.com/Sahaj-Chakka/IntegriSphere)
